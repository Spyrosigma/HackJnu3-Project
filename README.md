# Weapon-Detection-Web-App
How to setup ?
1) Clone this repo.
    ```bash
    git clone https://github.com/Spyrosigma/HackJnu3-Project.git
3) Install the required Libraries
    ```bash
    pip install -r requirements.txt
    ```
4) To run the web-app
    ```bash
    python manage.py runserver
    ```
